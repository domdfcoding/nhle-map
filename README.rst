=========
nhle-map
=========

.. start short_desc

**Map showing places on the National Heritage List for England.**

.. end short_desc

Dataset downloaded from the `Historic England Open Data Hub`_.  Licenced under the `Open Government Licence v3 (OGL)`_.

Installation
--------------

.. start installation

``nhle-map`` can be installed from GitHub.

To install with ``pip``:

.. code-block:: bash

	$ python -m pip install git+https://github.com/domdfcoding/nhle-map

.. end installation
